== Description ==

Sidebar Widget for your blog. Features South Florida Breaking News, Broward County News and Palm Beach County News. 

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `sun-sentinel.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Drag Sun-Sentinel Breaking News Widget to your Sidebar

== Frequently Asked Questions ==
- How do I upload sun-sentinel.php to the /wp-content/plugins/ directory? -
- Use your FTP manager, browse to the /wp-content/plugins/ directory of your blog and upload sun-sentinel.php to that folder.
